# Star-Delta-Forward-Reverse-Control
A comprehensive implementation of a Star-Delta motor starter with Forward/Reverse control. This repository contains both classical relay logic designs (simulated in CADe-SIMU/PC_SIMU) and modern PLC ladder logic implementations (developed in TIA Portal for Siemens S7-1200)

**Project Overview**
This project demonstrates the Star-Delta starter circuit, designed to mitigate high starting currents in induction motors while providing forward/reverse capability.

**Technologies Used**
Relay Logic: Designed and simulated in CADe-SIMU/PC_SIMU.

PLC Logic: Developed in TIA Portal (Siemens S7-1200) and tested in PLC SIM.

**Structure**
Ladder Diagram zip file: contains TIA portal files for the ladder diagram
Relay Logic Control: contains CADe-SIMU/PC_SIMU files
Images: contains images of the programs
